bool RandFuel::calcCombinations ( long  p_nX,

long *  p_nT,
double ***  p_ca,
double ***  p_ra 
)

Performs the following steps:

  1. Calculates factorial combinations of all fuel types,
  2. puts probabilities into *comb and spread rates into *ros
  3. fills **ca with factorial combinations of probabilities (from comb) for all rows and columns in test block ( p_nX x p_nY )
  4. Fills **ra with factorial combinations of spread rates (from ros) for all rows and columns in test block ( p_nX x p_nY )

void RandFuel::calcSpreadRates ( void  )
protected
  1. Allocates threads and m_maxRosArray array to store max spread rates from all blocks
  2. Divides the Number of Combinations (m_combs) into parts for each thread.
  3. Runs each thread and wait until they are all finished (hRandSyncEvent)
  4. Calculates Expected Spread Rates by Prob[i] X MaxSpread[i]

◆ recomputeSpread()

double RandFuel::recomputeSpread ( double *  p_harmonicRos)

Recomputes spread using the existing spread rate array m_maxRosArray after the user has run ComputeSpread().

RecomputeSpread() assumes the user has loaded only new fuel data, changing only the proportions of fuels on the landscape. All that is needed then is to recompute the probabilities of the fuel arrangement for weighting in the harmonic mean calculation.
