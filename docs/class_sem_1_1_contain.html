Detailed Description

Fire flank (half-a-fire) containment object.

Just the bare information to make a single simulation pass for half a fire is included. The fire is assumed to grow in an elliptical shape.

◆ calcUh()

bool Sem::Contain::calcUh ( double p,
double h,
double u,
double * d )
protected

Determines du/dh for a particular u, h, and p, and returns the value in d.

If there is a negative term under the radical, which is indicative of the ContainResources being overrun by the fire, m_status is set to Overrun and the function returns FALSE. Similarly if a sign change occurs in du/dh.

Parameters
[in]pFireline production rate (ch/h).
[in]hCurrent distance of free-burning fire head from the origin (ch).
[in]uCurrent angle from the fire origin to the point of active fireline construction.
[in]dAddress where the du/dh derivative is returned.
Return values
TRUEif ContainResources are not overrun and d has valid result.
FALSEif ContainResources are overrun, and m_status is set to Overrun.
