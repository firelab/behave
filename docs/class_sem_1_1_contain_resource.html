Detailed Description

A single fire containment resource unit that can be dispatched to a fire. Examples include an engine crew, line crew, bulldozer, helicopter, airtanker, etc.

If the resource is assigned to either the LeftFlank or RightFlank, its fireline production rate is applied in full to that flank. If the resource is assigned to BothFlanks, then half of its fireline production rate is applied to each flank. If the resource is assigned to NeitherFlank, it is not used.
