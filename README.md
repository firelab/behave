# Behave
A new implementation of the extended Rothermel model

## Dependencies
- [CMake](https://cmake.org/download/)
- [LLVM/Clang](https://clang.llvm.org/)
- [Doxygen](doxygen.nl)

## Building
```bash
make
```

## Creating Docs + Tags
```bash
make dev
```

## Testing

### Surface / Crown / Contain
```bash
make test
```

### Mortality
```
make test_mortality
```
